# my_final_project
# سیستم نمره دهی پیشرفته دانشجویان (Advanced Student Grading System)

این پروژه یک برنامه C++ برای مدیریت اطلاعات دانشجویان و محاسبه نمرات آن‌ها است. سیستم امکان ثبت دانشجویان جدید، محاسبه معدل (GPA) و تعیین وضعیت تحصیلی (قبول، مشروط، مردود) را فراهم می‌کند، همچنین آمارهای کلی کلاس را نیز نمایش می‌دهد.

## ویژگی‌ها

*   **افزودن دانشجو:** ثبت نام، شناسه، دروس، نمرات و واحدهای هر دانشجو.
*   **محاسبه خودکار GPA:** محاسبه معدل بر اساس نمرات و واحدهای دروس.
*   **تعیین وضعیت تحصیلی:** دسته‌بندی وضعیت دانشجو بر اساس GPA (موفق، مشروط، مردود).
*   **آمار کلاس:** نمایش میانگین GPA کلاس، بالاترین و پایین‌ترین GPA.
*   **گزارش جامع دانشجویان:** نمایش اطلاعات کامل دانشجویان شامل جزئیات دروس، GPA و وضعیت تحصیلی.

## ساختار کد

*   **`struct Course`**: نگهداری اطلاعات هر درس (نام، نمره، واحد).
*   **`struct Student`**: نگهداری اطلاعات دانشجو (نام، شناسه، دروس، GPA، وضعیت).
*   **`vector<Student> students`**: ذخیره‌سازی اطلاعات تمامی دانشجویان.
*   **توابع کمکی**:
    *   `calculateGPA`: محاسبه GPA دانشجو.
    *   `calculateStatus`: تعیین وضعیت تحصیلی.
*   **توابع اصلی**:
    *   `addStudent`: افزودن دانشجو جدید.
    *   `classStatistics`: نمایش آمار کلاس.
    *   `showAllStudents`: نمایش گزارش کامل دانشجویان.
*   **`main()`**: مدیریت منوی اصلی و فراخوانی توابع.

## نحوه اجرا

1.  **پیش‌نیاز:** کامپایلر C++ (مانند g++).
2.  **کامپایل:**
    ```bash
    g++ your_file_name.cpp -o student_system
    ```
    (نام فایل C++ خود را جایگزین کنید)
3.  **اجرا:**
    ```bash
    ./student_system
    ```
4.  **استفاده:** از طریق منوی تعاملی برنامه، عملیات مورد نظر را انتخاب کنید.

## منطق و هدف پروژه

هدف اصلی این پروژه، پیاده‌سازی یک سیستم ساده و کاربردی برای مدیریت داده‌های دانشجویان در محیط کنسول است. منطق اصلی برنامه بر پایه ساختاردهی اطلاعات (با استفاده از struct ها) و انجام محاسبات مربوط به نمرات و معدل (GPA) بنا شده است. این سیستم به کاربران اجازه می‌دهد تا به راحتی اطلاعات دانشجویان را وارد، مشاهده و تحلیل کنند.
